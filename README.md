# Medusa-eCommerce-Website
Medusa.js is an open-source eCommerce framework that allows developers to build headless eCommerce websites with ease.


# Medusa API
   cd my-medusa-store/backend
   
   npm start


# Storefront
   cd my-medusa-store/storefront
  
  npm run dev
